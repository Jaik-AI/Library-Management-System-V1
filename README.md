# Library-Management-System-V1
A Python-based library management system that allows users to add, view, search, delete, and manage book records using file handling.


## Features

- Add new books
- View all books
- Search books by name
- Delete books
- Count total books
- Prevent duplicate book entries
- Persistent storage using File I/O

## Technologies Used

- Python
- File I/O
- Functions
- Exception Handling
- Lists
- String Manipulation

## How to Run

```bash
python main.py
```
